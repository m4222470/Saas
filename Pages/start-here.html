File Structure

Understanding the project structure will help you navigate and customize your template.

dark-x-saas-template/ ├── index.html # Homepage ├── pages/ │ ├── primary-home.html # Primary Homepage │ ├── about.html # About Page │ ├── features.html # Features Page │ ├── pricing.html # Pricing Overview │ ├── pricing-single.html # Single Plan Page │ ├── contact.html # Contact Page │ ├── blog.html # Blog Listing │ ├── blog-post.html # Single Post │ ├── team.html # Team Listing │ ├── team-member.html # Team Member │ ├── careers.html # Careers Listing │ ├── careers-single.html # Job Details │ ├── integrations.html # Integrations │ ├── integration-single.html │ ├── 404.html # Error Pages │ ├── 401.html │ └── start-here.html # This Guide ├── src/ │ ├── styles/ │ │ └── main.css # Main Stylesheet │ └── scripts/ │ └── main.js # JavaScript └── assets/ └── images/ # Images

Customization

Personalize your template by modifying the CSS variables in main.css.

// Color Variables --color-bg-primary: #0a0a0f; --color-bg-secondary: #12121a; --color-bg-tertiary: #1a1a24; --color-accent: #6366f1; --color-accent-2: #f472b6; // Typography --font-family: 'Inter', sans-serif; --font-size-base: 16px;
Tip: You can also add custom CSS in the <style> tag of each HTML page for page-specific styles.
